## 1. 핵심 동작 — PTT 받아쓰기 (P1 DoD)

> "핫키를 누르고 말하면 현재 커서 위치에 한국어 텍스트가 들어간다"

| # | 테스트 항목 | 예상 결과 | 결과 |
|---|-----------|---------|------|
| 1.1 | 앱 실행 → 트레이 아이콘 표시 확인 | 트레이에 STT-AIO 아이콘 표시 | ☐ |
| 1.2 | PTT 핫키(`Ctrl+Alt+R` 기본) 누르는 동안 오버레이 표시 | "녹음 중" 오버레이 표시 | ☐ |
| 1.3 | 핫키 누르고 한국어 발화("안녕하세요") → 핫키 해제 | 커서 위치에 "안녕하세요" 입력됨 | ☐ |
| 1.4 | 메모장에서 1.3 반복 — 한글 손실/깨짐 없음 확인 | 원문과 동일한 텍스트 주입 | ☐ |
| 1.5 | VSCode 에디터에서 1.3 반복 | 정상 주입 | ☐ |
| 1.6 | Chrome 주소창에서 1.3 반복 | 정상 주입 | ☐ |
| 1.7 | 토글 모드 핫키로 녹음 시작 → 재핫키로 종료 → 주입 확인 | PTT와 동일하게 주입 | ☐ |
| 1.8 | 무음 또는 소음만 있을 때 → 빈 주입 없음 | 아무것도 주입 안 됨 | ☐ |
